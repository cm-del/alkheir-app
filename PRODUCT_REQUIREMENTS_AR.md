# وثيقة المتطلبات - تطبيق إدارة مزرعة دواجن تسمين

## الهدف
تطبيق أندرويد احترافي ثنائي اللغة (عربي/إنجليزي) لإدارة مزارع التسمين مع دعم العمل بدون إنترنت ومزامنة عند توفر الشبكة.

## الوحدات الرئيسية
1. إدارة الدورات والعنابر.
2. التسجيل اليومي (نفوق، علف، بيئة، ملاحظات).
3. تتبع الأوزان ومؤشرات الأداء.
4. المخزون (علف/دواء/مطهرات/مستهلكات).
5. المصروفات والإيرادات والربحية.
6. التنبيهات الذكية.
7. التقارير التصديرية (PDF ومشاركة واتساب).
8. إدارة المستخدمين والصلاحيات.

## مؤشرات الأداء المستهدفة
- FCR
- Mortality Rate
- Average Daily Gain
- Cost per KG
- Net Profit per Cycle

## متطلبات غير وظيفية
- Offline-first مع local database.
- مزامنة تدريجية conflict-aware.
- واجهة استخدام سهلة لغير التقنيين.
- دعم كامل RTL/LTR.
- نسخ احتياطي واستعادة.

## الصلاحيات
- مالك
- مدير
- عامل إدخال
- محاسب

## خطة التسليم
- الإصدار 1: النواة التشغيلية اليومية.
- الإصدار 2: التقارير الذكية والتنبيهات.
- الإصدار 3: تعدد المستخدمين + مزامنة متعددة الأجهزة.
